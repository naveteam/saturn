# Button

O componente `Button` é utilizado para permitir que o usuário possa realizar ações dentro da aplicação.

# Color

O `Button` pode receber a prop `color`. A variant `primary` tem uma função de mais destaque, ideal para executar ações e/ou mostrar informações fundamentais para o usuário. A variant `secondary` é ideal para funções de menores destaque.

<Canvas>
  <Story
    name='Color'
    parameters={{
      design: {
        type: 'figma',

      }
    }}
  >
    <ThemeProvider>
      <Button width={1 / 4}>Primary</Button>
      <Button width={1 / 4} color='secondary'>
        Secondary
      </Button>
    </ThemeProvider>
  </Story>
</Canvas>

# Variant

Em conjunto com a prop `color`, o `Button` pode receber também a prop `variant`. As variants podem criar botões preenchidos, apenas contornados ou apenas o texto.
