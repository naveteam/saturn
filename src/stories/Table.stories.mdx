# Regular Table

O componente `Table` permite que sejam criadas tabelas estilizadas para apresentação de dados na aplicação.

O tipo padrão de tabela é a `regular` que se diferencia do outro tipo disponível devido a borda externa que a compõe.

      type: 'figma',

    }
  }}>
    <ThemeProvider>
      <Table type="regular">
        <TableContent as='thead'>
          <TableData>
            <Checkbox />
            <Typography fontWeight={1} fontSize={3} lineHeight={3}>Content</Typography>
          </TableData>
          {['Teste' , 'Teste1'].map((header, index) => (
            <TableData key={index}>
              <Typography fontWeight={1} fontSize={3} lineHeight={3}>
                {header}
              </Typography>
            </TableData>
          ))}
        </TableContent>
        <TableContent>
          <TableRow>
            <TableData as='td'>
              <Checkbox />
              <Typography>Content</Typography>
            </TableData>
            <TableData><Typography>Content</Typography></TableData>

          </TableRow>
          
        </TableContent>
      </Table>
    </ThemeProvider>
  </Story>
</Canvas>
