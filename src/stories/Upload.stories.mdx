# Disabled

A prop `disabled` desabilita completamente o botão, impedindo seu uso.

  </Story>
</Canvas>

# Variant

O componente `Upload` pode receber também a prop `variant`. As variants podem criar botões preenchidos, apenas contornados, drag and drops ou upload de imagem com preview.

<Canvas>
  <Story
    name='Variants'
    parameters={{
      design: {
        type: 'figma',
        url: 'https://www.figma.com/file/O3bKxIcsj2rc1FNIRclJyT/Saturn-System?node-id=284%3A855'
      }
    }}
  >
    <SaturnProvider>
      <Upload caption='Upload Button (Primary)' variant='button' name='upload-button' />
      <Upload caption='Upload Button (Secondary)' variant='button-outlined' name='upload-button-outlined' />
      <Upload
        caption='Upload Drag &amp; Drop'
        description='Aqui vai uma pequena descrição sobre os formatos de arquivos.'
        variant='drag-drop'
        multipleFiles
        name='upload-drag-and-drop'
      />
      <Upload caption='Upload Image' variant='image' name='upload-image' />
    </SaturnProvider>
  </Story>
</Canvas>
